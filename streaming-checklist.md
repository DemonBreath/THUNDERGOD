## Gear — buy list

### Microphone (priority #1)

- [ ] Choose mic
  - [ ] **FIFINE K688** (~$50–80) — best budget dynamic, USB + XLR
  - [ ] **Deity VO-7U** (~$84) — strong dynamic, built-in limiter
  - [ ] **PreSonus Revelator Dynamic** (~$105) — step-up with mixer/loopback
  - [ ] **HyperX QuadCast 2** (~$130–150) — only if room is quiet (condenser)
- [ ] Buy headphones (prevent echo/feedback)
- [ ] Optional: boom arm (~$25–40)
- [ ] Test mic levels in OBS (not too quiet, not clipping)

### Webcam

- [ ] Choose webcam
  - [ ] **NexiGo N660P Gen 2** (~$60) — recommended: 1080p60 + autofocus
  - [ ] **Logitech C920x** (~$59–70) — classic, reliable, 1080p30
  - [ ] **NexiGo N680E Pro 4K** (~$71) — built-in ring light
  - [ ] **Razer Kiyo** (~$89) — built-in light, good for dim rooms
  - [ ] **Logitech C922x** (~$99) — step-up 1080p60
- [ ] Mount webcam (monitor clip or tripod)
- [ ] Frame shot in OBS (face cam size + position)
- [ ] **Use external mic, not webcam mic**

### Backdrop

- [ ] Choose backdrop style
  - [ ] **Dark curtain + tension rod** (~$20–40) — simple, pro look, no green screen
  - [ ] **Arozzi Green Screen** (~$50) — pop-up, stores away
  - [ ] **Neewer 5×7ft collapsible** (~$70) — green + blue sides
  - [ ] **EMART retractable panel** (~$80–110) — pull-up from case
- [ ] Hang / set up backdrop
- [ ] Check how it looks on camera

### Lighting

- [ ] Get a key light (if webcam has no built-in light)
  - [ ] Small LED panel (~$25–40), or
  - [ ] Ring light (often bundled with webcam)
- [ ] Position light in front of face (not behind you)
- [ ] Test on camera — no harsh shadows on face

### PC storage

- [ ] Check PC for free **M.2 slot** (motherboard manual or Device Manager)
- [ ] Note PCIe version (Gen4 vs Gen5)
- [ ] Choose SSD
  - [ ] **Crucial P3 Plus 2TB** (~$110–130) — best value for space
  - [ ] **WD Black SN850X 2TB** (~$130–160) — fast + spacious
  - [ ] **Samsung 990 EVO Plus 2TB** (~$150–180) — reliable step-up
  - [ ] **Crucial P510 1TB** (~$85) — tight budget option
- [ ] Buy and install SSD
- [ ] Set OBS recording folder to new drive
- [ ] Confirm free space before each stream (aim for 50+ GB free)

## Team — video editor (friend)

- [ ] Ask friend if they're interested
- [ ] Confirm their editing software (DaVinci, Premiere, CapCut, etc.)
- [ ] Agree on compensation model
  - [ ] Portfolio + credit (early)
  - [ ] Per video ($25–75 short / $75–200 long)
  - [ ] Rev share later (write it down if yes)
- [ ] Do **one test clip** (send 10 min footage → get 45 sec clip back)
- [ ] Set up shared folder (Google Drive / Dropbox)
  - [ ] `/raw` — full OBS recordings
  - [ ] `/clips` — finished exports
  - [ ] `/assets` — logo, fonts, colors, music
- [ ] Send brand assets (THUNDERGOD palette, logo, tagline)
- [ ] Send 2–3 style reference videos
- [ ] Define deliverables per week
  - [ ] 2 Shorts / TikToks
  - [ ] 1 YouTube video (weekly or biweekly)
- [ ] Define your job after each stream
  - [ ] Upload raw file same night
  - [ ] Send 3–5 timestamps of best moments
- [ ] Credit editor in every video description

## Team — stream manager (while live)

- [ ] Ask friend if they're interested
- [ ] Make them a **Twitch mod** (sword icon on channel)
- [ ] Share chat rules (timeout vs ban)
- [ ] Share canned answers for common questions
  - [ ] What is THUNDERGOD?
  - [ ] What is Alexworld?
  - [ ] Discord link
  - [ ] Schedule
  - [ ] GitHub link
- [ ] Set interrupt rules
  - [ ] **Interrupt you for:** audio dead, stream offline, big dono, guest arriving
  - [ ] **Don't interrupt for:** routine chat, !commands, small stuff
- [ ] Create private Discord voice channel for urgent pings
- [ ] Shared doc for clip timestamps during stream
- [ ] Agree on compensation
  - [ ] Volunteer + credit (early)
  - [ ] Per stream ($15–40)
  - [ ] Monthly retainer ($50–150) if regular
- [ ] Run **one test stream** with them managing (90 min)
- [ ] Debrief after test stream — what worked, what to change

### Stream manager — live night run-of-show

**Before stream**
- [ ] Confirm stream is scheduled / title is set
- [ ] Post "going live" on Discord + social
- [ ] Check OBS is live and audio looks healthy on dashboard
- [ ] Slow mode ready if chat gets fast

**During stream**
- [ ] Moderate chat (timeouts, bots, vibe)
- [ ] Answer repeat questions
- [ ] Flag important messages to streamer
- [ ] Mark clip timestamps in shared doc
- [ ] Watch bitrate / dropped frames
- [ ] Run chat commands when asked

**End of stream**
- [ ] Send clip timestamp list to editor
- [ ] Thank raiders / hosts
- [ ] Confirm recording saved to shared folder
- [ ] Quick debrief with streamer

- [ ] Webcam framed and lit (or voice-only decision made)

- [ ] Water nearby
- [ ] Stream title set
- [ ] Discord link works
- [ ] Tell 3 people when you're going live
- [ ] Manager and/or editor know their roles for tonight
